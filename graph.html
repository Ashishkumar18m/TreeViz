Graph Visualizer - Help Guide

Getting Started

This tool allows you to draw custom graphs and visualize Depth-First Search (DFS) and Breadth-First Search (BFS) algorithms.

Drawing Graph:

  • Create Nodes: Click anywhere on the left canvas to create nodes
  • Create Edges: Click on a node and drag to another node to create a connection
  • Delete Nodes: Right-click on any node to delete it
  • Assign Values: Use the input fields below to assign custom values to nodes

Visualization Steps:

  • Select Algorithm: Choose either BFS or DFS
  • Choose Starting Node: Select where the traversal should begin
  • Start Visualization: Click "Start Visualization" to begin
  • Step Through: Use Next/Previous buttons to move through steps
  • Auto Play: Click "Auto Play" for automatic step-by-step visualization

Algorithm Details:

Breadth-First Search (BFS):

  • Explores all neighbors at the present depth before moving to next level
  • Uses a Queue data structure
  • Finds the shortest path in unweighted graphs

Depth-First Search (DFS):

  • Explores as far as possible along each branch before backtracking
  • Uses a Stack data structure (recursive or explicit stack)
  • Good for cycle detection and topological sorting

Tips:

  • Use "Sample Tree" and "Sample Graph" buttons for quick examples
  • You can draw disconnected graphs
  • Cycles are allowed - draw edges to create loops
  • Node values can be letters, numbers, or short words
